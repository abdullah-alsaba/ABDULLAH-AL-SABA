Description

An interactive color selection web app where users can generate and experiment with different color combinations in real-time. Focused on UI interaction and JavaScript logic.

Challenges Faced

Keeping the UI responsive while updating colors live, and making sure generated palettes remain readable with accessible contrast.

Future Improvements

Add palette saving/export, introduce accessibility checks (contrast warnings), and provide preset themes for quick exploration.
