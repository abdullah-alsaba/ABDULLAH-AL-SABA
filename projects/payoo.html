Description

A digital banking UI concept named PAYOO. It simulates a modern mobile banking experience with account overview, transactions UI, and a clean financial dashboard layout.

Challenges Faced

Designing a finance-focused UI that stays clear and readable while showing multiple data blocks, and maintaining consistent spacing across different screen sizes.

Future Improvements

Add interactive filtering for transactions, improve accessibility for contrast and focus states, and expand the dashboard with additional account insights.
